# Scoring rubric (0–100)

Use this rubric for **Fast Audit** and **Standard Audit**.

## Dimensions
| Dimension | Weight | What to look for |
|---|---:|---|
| UX/UI | 15 | clarity, hierarchy, nav, accessibility basics, responsiveness |
| Conversion | 20 | value prop, CTA, trust, offer clarity, friction |
| SEO | 20 | information architecture, indexability signals, internal links, metadata |
| AEO | 10 | FAQ, question-answer pages, entities, snippet-friendly structure |
| GEO | 10 | consistent entities, citability, structured explanations |
| Site structure quality | 10 | sitemap/page map coherence, depth, findability |
| Template-ability | 15 | reusable sections/components, clear patterns |

## Labels
- **P0/P1/P2** for recommendations
- **confidence: high/medium/low** for hypotheses

## Acceptance rule
- Include in final library if **score >= 70** AND adds pattern diversity.
