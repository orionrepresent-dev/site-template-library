# Site Template Library (UX/SEO/AEO/GEO/Conversion)

This repository is generated from reverse engineering **200** high-quality public websites (50/50 BR & Global) across these niches:
- Professionals (Profissionais liberais)
- Restaurants (Restaurantes)
- Stores / E-commerce (Lojas)
- Services (Prestação de serviços)

## What you get
- A curated benchmark list (Google Sheet) with discovery source, scoring, and report links
- Reusable **page templates** and **component patterns** extracted from the best sites
- Pattern playbooks for UX, SEO, AEO/GEO, and conversion

## Workflow (high level)
1. Discover candidate sites (hundreds) per niche/geo
2. Run **Fast Audit (quick)** to score and shortlist
3. Run **Standard Audit** on the final 200
4. Mine patterns → generate templates/components → commit to this repo

## Repo structure
- `templates/` — niche templates (pages/components/content models)
- `patterns/` — cross-template playbooks (UX/SEO/AEO/GEO/Conversion)
- `benchmarks/` — benchmark site list exports
- `reports/` — optional markdown report exports (if you choose to store here)
